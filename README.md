# Modern Snake

## Game Mode
* **one player**: eat every fruits
* **two or three players**: the game ends when every player is dead
* if this is too easy, you can play with **obstacles**
* maybe there is also **"secret" mode**

## How to play
* **player1**: arrows
* **player2**: WASD / ZQSD
* **player3**: IJKL
